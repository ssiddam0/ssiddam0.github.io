SWTL JS Chapters 11-20 Samples

Chapter 11 - Comparison operators

===, !==; === , use it to compare two things to see if they're equal.
!==, is the opposite of ===, It means is not equal to.

Chapter 12 - if..else..else if

use if to specify a block of code to be executed, if a specific condition is true
use else to specify a block of code to be executed, if the same condition is false

Chapter 13 - Compound Statements

Two or more conditions can be tested in one if statement.
&& - equivalent of 'and' in English
|| - equivalent of 'or' in English

Chapter 14 - Nested if

Nested if statements, meaning one if block inside of another, allows you to test multiple criteria and increases the number of possible outcomes.

Chapter 15 - Arrays

An array is a set of data elements stored under the same name.
var array_name = [item1, item2, ...];
var primaryColors = new Array("red", "green", "blue");

Chapter 16 - Push/Pop Array Elements

The pop() method returns the value that was "popped out": secColors.pop('purple');
The push() method adds a new element to an array (at the end): secColors.push('purple');

Chapter 17 - More Array Methods

Use the shift method to remove an element from the beginning of an array.
To add one or more elements to the beginning of an array, use the unshift method.
Use the splice method to insert one or more elements anywhere in an array, while optionally removing one or more elements that come after it.

Chapter 18 - for loops

Loops can execute a block of code a number of times. Loops are handy, if you want to run the same code over and over again, each time with a different value.

Chapter 19 - for loops..loop interrupts

When the break statement is used in a loop, it breaks the loop and continues executing the code after the loop (if any).

Chapter 20 - Nested for loops

Loops with in loops, you can have as many levels of nesting as you like.
